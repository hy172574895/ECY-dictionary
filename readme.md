# Installation, this a plugin of ECY.
Required
> Plug 'hy172574895/EasyCompleteYou'

Use Plugin-manager
> Plug 'hy172574895/ECY-dictionary'

and then 

```
:ECYInstall dictionary
```

中国用户可以使用这个镜像，很快很多很多！
> Plug 'https://gitee.com/Jimmy_Huang/ECY-dictionary'
